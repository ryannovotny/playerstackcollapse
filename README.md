# Player Stack Collapse
A RuneLite plugin to collapse right-click menus for multiple players into sub-menus.

<img width="542" height="198" alt="image" src="https://github.com/user-attachments/assets/7c79502c-3a68-4d9a-aaf0-d72404531546" />

## Features
* Enabling/disabling collapsed menus
* Specifying the number of players needed to collapse menus
* Automatic disabling of collapsed menus if in the Wilderness or PvP zones
* Only enabling collapsed menus in high-traffic areas (Grand Exchange, Wintertodt, GotR, and Pest Control)
